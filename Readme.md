# Java Calculator

A console-based calculator application developed in Java.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Division by zero protection
* Menu-driven interface
* Clean code structure
* Git workflow implementation

## Technologies

* Java
* Git
* GitHub

## Project Structure

```text
java-calculator/
│
├── src/
│   ├── Main.java
│   │
│   ├── services/
│   │   └── CalculatorService.java
│   │
│   └── utils/
│       └── Menu.java
│
├── docs/
├── README.md
└── .gitignore
```

## How To Run

Compile:

```bash
javac src/Main.java src/services/CalculatorService.java src/utils/Menu.java
```

Run:

```bash
java -cp src Main
```

## Sample Usage

```text
===== JAVA CALCULATOR =====

1 - Addition
2 - Subtraction
3 - Multiplication
4 - Division
0 - Exit

Choice: 1

First Number: 10
Second Number: 20

Result: 30.0
```
